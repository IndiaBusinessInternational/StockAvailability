Backend Connection
This app stores data in your Google Sheet through a Google Apps Script web app. Deploy IBIStockAvailability_GAS.gs (Deploy → New deployment → Web app → Execute as Me, Access Anyone), then paste the /exec URL below.
Import Product Names
Upload a List of Product Names file (.xlsx, .xls or .csv). The product-name column is detected automatically. Only names are added — stock counts stay blank for you to fill in later.
